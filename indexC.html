O que significa Compssão de Dados?

A compactação de dados é o processo de modificar, codificar ou converter a estrutura de bits dos dados de maneira a consumir menos espaço no disco. Permite reduzir o tamanho do armazenamento de uma ou mais instâncias ou elementos de dados. A compactação de dados também é conhecida como codificação de origem ou redução da taxa de bits.

O que é Compressão?

A compactação de dados funciona através de várias técnicas de compactação e soluções de software que utilizam algoritmos de compactação de dados para reduzir o tamanho dos dados. Uma técnica comum de compactação de dados remove e substitui elementos e símbolos de dados repetitivos para reduzir o tamanho dos dados.
