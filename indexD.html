Descompressão de Dados

________________________________

O que significa descompressão?

A descompactação é o processo de restauração dos dados compactados à sua forma original. A descompactação de dados é necessária em quase todos os casos de dados compactados, incluindo compactação com perdas e sem perdas. Semelhante à compactação de dados, a descompactação de dados também é baseada em diferentes algoritmos. A descompactação é considerada importante, pois os dados compactados precisam ser restaurados ao estado padrão para uso. A descompressão é amplamente utilizada em comunicações de dados, multimídia, áudio, vídeo e transmissões de arquivos.

O que é descompressão?

O aplicativo necessário para a descompactação depende em grande parte de como os dados foram compactados em primeiro lugar. Na maioria dos casos, o software ou aplicativo necessário para descompactação de dados também vem com o aplicativo ou software usado para compactação de dados. No caso de compactação sem perdas, os dados originais são obtidos sem nenhuma perda na descompactação. Este não é o caso da descompressão para métodos com perdas, pois pode haver perda nos dados originais, embora possa não afetar o receptor. A descompactação de dados ajuda a remover as complicações adicionadas pela compactação de dados.
